# 壹·言

> 言者，心聲也。符者，意之形也。合而言之，謂之語句。

## 字符

夫文字之始，結繩記事。其後倉頡造字，以形代事。今之程式，亦循此理——以**字符**載信息，以**語句**述邏輯。

字符有四類：

- **字**（letters）：`甲乙丙丁` 之類
- **數**（digits）：`一二三` 或 `123`
- **符**（symbols）：`。，；：「」` 之類
- **空**（whitespace）：空格、換行、縮進

## 語句

語句者，完整之意也。以句號「。」結之，謂之一句。

```
吾有一言。
```

此乃最短之語句。言者何？符號也。符號者，名之表也。

## 註解

註解者，備忘也。不入執行，唯供閱覽。以 `註` 起，`畢` 止。

```
註：以下為示例畢
吾有一言。註：此乃註解畢
```

## 凡符號

文言編程所用之符號，多襲古意：

| 文言 | 今譯 |
|:----:|:----:|
| `吾有` | let / const |
| `名之` | = |
| `之` | 的 / 之 |
| `其` | it / 其 |
| `是謂` | return |

## 小結

> **本章要旨**：字符組合，語句成章。註解備忘，凡例須明。

下章論**數**——天地之間，莫非數也。
