# 說明

## 鍵盤操作

- `←` / `→`：上一章 / 下一章
- `V`：切換豎排 / 橫排
- 滾輪：豎排模式下，**上下滾輪映射為左右滾動**（中文豎排習慣）
- 拖動底部滑塊：跳轉到指定章
- 點擊頂部 tab：切換章

## 排版說明

- **豎排模式**（默認）：自右至左，自上而下，仿古籍陰符
- **橫排模式**：現代閱讀習慣，視個人偏好切換

## 字體

本主題優先使用系統楷體：

- macOS / iOS：`STKaiti`、`STSong`
- Windows：`KaiTi`、`SimSun`
- Linux：`Noto Serif SC`、`Source Han Serif SC`（需自行安裝）

若需更高品質，建議自行下載思源宋體並放入 `docs/.vuepress/public/fonts/`，然後於 `theme/styles/index.scss` 開頭加載 `@font-face`。
