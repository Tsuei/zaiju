# 伍·判

> 物有是非，事有可否。程式之道，亦在於判。

## 若

判之最簡者，`若...則...也`：

```
若「大於十」者。則「甲」是矣。云云。
```

## 否

若不若，則為否：

```
若「甲」大於十者。
  吾有一言。名之曰「乙」。昔之「乙」者「甲大」。今「乙」是矣。
若非。
  吾有一言。名之曰「乙」。昔之「乙」者「甲小」。今「乙」是矣。
也。
```

## 多判

層層遞進：

```
若「甲」大於十者。
  ...
若「甲」大於五者。
  ...
若非。
  ...
也。
```

## 比之諸符

| 文言 | 今譯 |
|:----:|:----:|
| `大於` | > |
| `小於` | < |
| `等於` | == |
| `不等於` | != |
| `且` | && |
| `或` | \|\| |
| `非` | ! |

## 小結

> **本章要旨**：判者，分流也。分之得當，萬緒可理；分之失當，寸步難行。

下章論**循**——往復循環，周而復始。
