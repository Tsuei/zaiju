# 柒·函

> 函者，式也。執一而行萬，萬變歸一。

## 定義函數

```
吾有一術。名之曰「倍」。欲行是術。必先得二數。曰「甲」。曰「乙」。
  是謂「甲」乘「乙」之術也。
是術之義畢。
```

## 調用

```
吾有一數。名之曰「果」。昔之「果」者倍之於三以四。今「果」是矣。
```

「果」今為十二。

## 遞迴

函可自調，謂之遞迴：

```
吾有一術。名之曰「階」。欲行是術。必先得一數。曰「甲」。
  若「甲」小於二者。
    乃得一以「甲」。
  若非。
    減一以「甲」。乃得「甲」乘階之於「甲」之術也。
是術之義畢。
```

此即階乘。

## 函之妙

函者，抽象之器也。執一術而用於萬事，不復重述其詳。善用函者，程式簡而意深。
