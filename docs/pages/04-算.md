# 肆·算

> 算者，數之相推也。加減乘除，因果由是。

## 四則

```
吾有二數。名之曰「甲」「乙」。
昔之「甲」者十。今「甲」是矣。
昔之「乙」者三。今「乙」是矣。

吾有一數。名之曰「和」。昔之「和」者「甲」加「乙」。今「和」是矣。
吾有一數。名之曰「差」。昔之「差」者「甲」減「乙」。今「差」是矣。
吾有一數。名之曰「積」。昔之「積」者「甲」乘「乙」。今「積」是矣。
吾有一數。名之曰「商」。昔之「商」者「甲」除以「乙」。今「商」是矣。
```

## 自增自減

變可自增一，謂之「加一自增」：

```
加一以「甲」。
```

亦可減一：

```
減一以「甲」。
```

## 模與整除

求餘：

```
十除以三餘一。
```

## 優先

算有先後，遵古制：先乘除，後加減。欲改其次，以括弧「（）」括之。

```
二乘（三加四）等於十四。
```

## 小結

> **本章要旨**：算雖簡，用則無窮。熟此四則，可馭萬機。

下章論**判**——是非曲直，皆由心定。
