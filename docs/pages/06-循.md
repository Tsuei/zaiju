# 陸·循

> 道生一，一生二，二生三，三生萬物。循者，萬物之機也。

## 為之

「為」者，遍歷之謂。設其起止，依序而進：

```
吾有一數。名之曰「甲」。昔之「甲」者〇。今「甲」是矣。
為「甲」十。
  加一以「甲」。
也。
```

此即計數自一至十。

## 恆為

不知終始，唯循不息：

```
恆為之。
  若「甲」大於百者。乃止。云云。
  加一以「甲」。
也。
```

## 遍歷

遍歷序列，逐一過之：

```
吾有一列。名之曰「眾」。昔之「眾」者「甲」「乙」「丙」。今「眾」是矣。
凡「眾」中之「子」。
  書之。
也。
```

## 小結

> **本章要旨**：循者，複之行也。知止而後能定，知返而後能安。

下章論**函**——抽象萬端，歸於一處。
