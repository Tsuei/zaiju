# 叁·變

> 數者，定也；變者，活也。以定馭活，萬事可成。

## 變之為名

變者，有名之器，納數以存。其名為符，其值為實。

```
吾有一數。名之曰「天」。
```

此宣告一變，名「天」，初值未設。

## 賦值

賦值者，授值於變也：

```
昔之「天」者一。今「天」是矣。
```

意即：「天」之值，現為一。

## 易值

變之名恆，其值可易：

```
吾有一數。名之曰「甲」。昔之「甲」者五。今「甲」是矣。
昔之「甲」者十。今「甲」是矣。
```

此「甲」先為五，後為十。

## 變之引

用變之時，徑稱其名：

```
吾有一數。名之曰「甲」。昔之「甲」者五。今「甲」是矣。
吾有一數。名之曰「乙」。昔之「乙」者「甲」加三。今「乙」是矣。
```

「乙」之值，今為八。

## 小結

> **本章要旨**：變為器，數為實。器有定名，實可更易。善用變者，程式活矣。

下章論**算**——以變馭數，運算成章。
