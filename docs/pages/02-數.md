# 貳·數

> 天地之大，萬物之眾，皆可計之以數。

## 數之為物

數有三種：

1. **整數**（int）：`一二三` 或 `-一二三`
2. **浮數**（float）：`一點五` 或 `1.5`
3. **複數**（complex）：罕用，茲不贅

## 書數之法

文言數字，承古制：

```
零 一 二 三 四 五 六 七 八 九 十
百 千 萬 億 兆
```

亦可用阿拉伯數字，二者皆通：

```
吾有一數。名之曰「甲」。昔之「甲」者。今「甲」是矣。
```

## 算數之基

四則運算，孩童皆知：

| 算 | 符 | 例 |
|:--:|:--:|:--:|
| 加 | `加` | 三加五 |
| 減 | `減` | 八減三 |
| 乘 | `乘` | 二乘四 |
| 除 | `除` | 十除以二 |

## 餘數

除之不盡，謂之餘：

```
十除以三餘一。
```

## 小結

> **本章要旨**：數雖無情，最為有信。一加一恆等於二，雖萬世不易。

下章論**變**——數居其所，動則有變。
